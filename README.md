# Math Quiz Game – C++

A console-based math quiz game where the player answers randomly generated questions.

## Features
- Choose number of questions (1-20)
- Choose difficulty level: Easy, Mid, Hard, Mix
- Choose operation type: Add, Sub, Mul, Div, Mix
- Tracks right and wrong answers
- Supports multiple games in one session

## Built With
- C++
